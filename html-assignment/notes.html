List of HTML Tags

The following are 5 HTML tags with their descriptions:

1. <p> </p>
It is a block-level tag that is used for creating

paragraphs

and separation between of them
2. <q> </q>
It is an inline tag used to make a quotation in text.
3. <em> </em>
It is an inline tag and is used to add emphasis to text.
4. <abbr> </abbr>
It is an inline tag that is used for acronyms or abbreviation like the program abbreviation IMD
5. <strong> </strong>
It is an inline tag that is used to put more emphais on text by making it bold
